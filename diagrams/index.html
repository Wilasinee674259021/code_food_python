➕ Sequence: POST /foods

Create Food — Pydantic validate → INSERT RETURNING id

Pydantic validate body ก่อน route handler ทำงาน — ถ้า field ผิด FastAPI ตอบ 422 ทันที · Food model ใช้ INSERT ... RETURNING id แล้ว cursor.fetchone()["id"] แทน cursor.lastrowid

⚡ Dependency Injection

Depends(get_db) — FastAPI inject psycopg2 connection อัตโนมัติ

Dependency Injection คือการที่ FastAPI สร้าง/จัดการ dependency ให้อัตโนมัติ · get_db() เป็น Generator function ที่ yield connection แล้ว finally: conn.close() — รับประกันว่า connection ปิดทุกครั้งแม้เกิด error
